Situation Brief
Situation: Cruise FL320. SCCM calls the flight deck: thick smoke in the mid-cabin, source unknown. MASTER CAUTION with single chime. Smoke can be fatal within minutes if recirculated. This scenario uses the QRH SMOKE/FUMES procedure.

Job: Don O2 masks immediately. Stop smoke recirculation (RECIRC FANS OFF). Identify source methodically. Assess: if source NOT 100% isolated — land immediately. No compromise.

ATC Protocol: Starts as PAN PAN (investigating source). Upgrades to MAYDAY if source not isolated within QRH time window.
