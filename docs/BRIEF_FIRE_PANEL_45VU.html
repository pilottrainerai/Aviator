Workflow

Phase 1

Vector Recreation

Create accurate vector artwork from the reference images provided above.

  • Panel outlines and section dividers
  • All pushbutton outlines — FIRE pb, AGENT pbs, TEST pbs, APU variants
  • Guard outlines including hinge tab and bottom pull-tab shape
  • Indicator cell outlines (SQUIB / DISCH) on each pushbutton
  • All labels, engravings, legends — exactly as reference
  • Vertical FIRE bar on each engine section left edge
  • Screw head positions
Output: Editable source file (AI / Figma) · SVG export
Phase 2

3D Modelling

Realistic 3D model based on vector artwork and specifications above. Reference model available: eng1_left_panel_BEST.blend (ENG 1 section).

  • Panel body — correct thickness, edge chamfer, screw recesses
  • ENG 1 FIRE pb body + face cells + protrusion
  • ENG 2 FIRE pb — identical to ENG 1
  • APU FIRE pb — same mechanism, larger dimensions
  • Guards — correct wall thickness, hinge knuckle, pull-tab — each guard as separate object
  • AGENT 1 + AGENT 2 pbs × 2 engine sections = 4 total
  • APU AGENT pb × 1
  • TEST pb × 3
  • Screw fasteners × 4
  • All indicator cells as distinct geometry — lens inset into pb face
  • Teal ring bezel around each AGENT pb
Every movable or physically separate component must be a separate object. Guards are separate from pb bodies. SQUIB and DISCH cells are separate from each other and from the pb body.
Output: Native Blender file (.blend) · FBX export · glTF/GLB export

Modelling Standards

Geometry & Realism
Topology
Clean edge flow — no unnecessary poles or triangles on visible surfaces
Edge bevels
Consistent — match reference dimensions; no sharp edges on moulded parts
Appearance goal
Controls should look manufactured — not computer-generated
Avoid
Perfectly flat pb faces · Exaggerated bevels · Guard walls too thick · Oversized screw heads · Sharp corners on any moulded element
